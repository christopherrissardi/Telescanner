#♟️ Hostscan

Ferramenta básica em shell para scanear hosts ativos na rede. Quando encontra um host, ele cita que o host está ativo.
Atualmente existe milhares de ferramentas que fazem esse mesmo processo e de forma 1000% mais eficiente, porém as vezes precisa fazer alguma adaptação em algum contexto voltado a segurança e esse script se torna útil em casos específicos.

Caso queira testar e/ou fazer alguma modificação, fique totalmente a vontade.

Projeto simples e prático.
